Workforce • Platform

ReachX-AI

ReachX-AI is designed as a workforce management and matching platform, with a focus on blue-collar and operations roles. It connects employers, dormitories, and workers in a structured way.

  • Manages workers, employers, dormitories and placements.
  • Central dashboard UI for status and matching.
  • Backed by Supabase, Jarvis, Cloudflare Workers and CSV pipelines.

Data & logic

What ReachX-AI tracks

Core entities

  • Workers (profiles, skills, status).
  • Employers (roles, requirements, locations).
  • Dormitories / housing.
  • Assignments and history.

Automation & UI

  • CSV → Supabase pipelines for worker data.
  • Cloudflare Workers for API endpoints.
  • Custom HTML/JS UI for employers, workers and dormitories.
