AION-ZERO

AION-ZERO is the meta-layer over Jarvis, designed to coordinate which commands, scripts and automations run across different AOGRL projects.

  • Holds “commands” and tasks in database tables.
  • Delegates execution to Jarvis and other agents.
  • Intended as the source of truth for what is running where.

What AION-ZERO coordinates

Projects

  • EduConnect — heartbeats, worker tasks, reporting jobs.
  • ReachX-AI — data refresh, UI checks, autopilot runs.
  • OKASINA & Deliveries — status and patch jobs.

Execution

  • Stores planned tasks in tables (e.g. Supabase).
  • Jarvis polls and executes them.
  • Logs outcomes back into project-specific tables.
