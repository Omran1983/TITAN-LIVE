================================================================================
TITAN COMPLIANCE SUITE (PRIVATE SECTOR) - DELIVERABLES PACKAGE
================================================================================

This folder contains all the assets you need to sell and deliver the compliance product.

1. SALES ASSETS (For getting clients)
-------------------------------------
> hr_compliance_pitch.html     : The Main Sales Landing Page (Upload to Vercel/Netlify).
> pitch.css                    : Styles for the sales page.
> logo_aogrl.png               : The Company Logo used in documents.
> ComplianceOps_Sample_Report.pdf : PROOF DOCUMENT. Send this to show clients what they get.

2. CLIENT DELIVERABLES (For paying customers)
---------------------------------------------
> templates/HR_01_Employment_Contract.md   : Standard Inspector-Ready Contract.
> templates/HR_02_Disciplinary_Letters.md  : Warning Letter Templates.
> templates/HR_03_Leave_Register.md        : Attendance/Leave Excel Structure.
> ComplianceOps_Agreement.pdf  : The Service Agreement (Generated per client).

3. OPS TOOLS (For you)
----------------------
> generate_sample_pdf.py       : Run this to regenerate the Sample Report.
> generate_agreement_pdf.py    : Run this to generate a new Client Agreement.
> generate_finance_docs.py     : Run this to generate Invoices/Receipts.
> generate_eml.py              : Run this to create the Deployment Email draft.
> Ops_Templates_And_Scripts.md : Your Script Book (Cold Emails, Flyers, LinkedIn Posts).

4. HOW TO DEPLOY
----------------
1. Upload `hr_compliance_pitch.html` + `pitch.css` + `logo_aogrl.png` to your hosting.
2. Send the "Deployment Email" (generated by generate_eml.py) to your prospects.
3. When a client signs up, use the scripts in Part 3 to generate their documents.
