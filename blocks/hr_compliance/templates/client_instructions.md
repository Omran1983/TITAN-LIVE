### TITAN HR Compliance – Data Submission Instructions

Thank you for using our HR Compliance Service.

Please follow the steps below:

1️⃣ Download the attached CSV template
2️⃣ Fill in employee details (one row per employee)
3️⃣ Save the file as:
```
employees_<CompanyName>_<Month>.csv
```

4️⃣ Email it back to:
```
compliance@yourdomain.com
```

Email subject:
```
CID: <YourClientID> | MONTH: YYYY-MM
```

You will receive your compliance report within **24–48 hours**.

Your data is processed locally and used **only** for compliance validation.

---

### Column Explanations

| Column | Description |
| :--- | :--- |
| `employee_name` | Full employee name |
| `salary` | Monthly basic salary (Rs) |
| `performance_status` | Good / Satisfactory / Excellent |
| `increment_proposed` | Yes / No |
| `hire_date` | YYYY-MM-DD |
| `job_grade` | Internal grade or category |
| `leave_taken_annual` | Days taken this year |
| `leave_taken_sick` | Sick leave days |
| `overtime_hours` | Monthly overtime hours |
| `last_increment_date` | YYYY-MM-DD |

**If a field doesn’t apply, leave it blank.**
